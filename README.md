
# ❗ 데이터 모델링의 개념(Data Modeling) 을 알고자 한다 ❗

> ### ERD : [DRAW.IO](https://app.diagrams.net/)

> ### SQL = MySQL

> ### 3가지 모델링

* #### 개념적 모델링: 개체와 개체들 간의 관계에서 ER다이어그램을 만드는 과정
* #### 논리적 모델링: ER다이어그램을 사용하여 관계 스키마 모델을 만드는 과정
* #### 물리적 모델링: 관계 스키마 모델의 물리적 구조를 정의하고 구현하는 과정
